HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is one of three cornerstones used in web development.
  • It creates structure in a webpage, which is made up of elements that work together to provide content and basic framework for all pages on the website.
  • Elements can be used to describe the type of content on a webpage (i.e.: image elements, text elements, header elements, and list elements).
  • Elements can also be used to divide content on a page and describe content to browsers and other web developers (i.e.: meta-related elements, content division elements, and semantic elements).
  • HTML stands for Hypertext Markup Language.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
